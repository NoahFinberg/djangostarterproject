Welcome to the Django Starter App (Authorization + Postgres)

Why use starter projects?

How long does it usually take you to setup user authentication, a postgres database, and a nice Bootstrap template when you're building a new web application?

For me, it usually took 2-3 hours of setup work. But it doesn't make sense to do the exact same setup every time you want to create a new Django project. Using Django in Minutes starter projects, you can have all of this setup in under 5 minutes!
